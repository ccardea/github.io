## Simple Time Tracker  

### History
- Designed for a single user running on Linux.
- Runs in a separate terminal while you work.
- Easy to configure.

### Current Status
Unused. Looking at this project after 4 years, its clear that this was just an exercise and not really intended for production. It works, but the user interface is clumsy and its utility is limited by the fact that there are no pre-defined reports. Also, it's designed to track multiple activities within projects, which is too much detail.  
[Project Web Page](https://ccardea.github.io/stt)  
[One Time Reports](https://colab.research.google.com/drive/1Q1hwEd0YKXMVSqGoksvXiFHHt1b7UmYx?usp=sharing)


### Plans For The Future
This is a candidate for migrating to Rust as an exercise. To be really useful it should have a user interface that runs in a browser and reports that can be customized through the UI.

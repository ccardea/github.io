## Hangman

### History
This project started with an MIT course assignment to implement a Hangman game in Python. I later went beyond the course assignment and added features including:
- An Assistant to help the player,
- The Assistant can play the game without a human player,
- Keeps statistics.

I then created a notebook that includes screenshots from the game, along with statistics and charts that show how the Assistant performs in both Interactive Assisted and Machine Only modes. The notebook helps to create a narrative and the charts are colorful and easy to read.

### Current Status
